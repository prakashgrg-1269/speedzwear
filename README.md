# 🥾 SpeedzWear — Shoe Store Management System

A Python CLI-based shoe store management system built as a coursework project.

## What it does
- 📋 Display all available shoes in the inventory
- 🛒 Purchase shoes and automatically generate a customer invoice
- 📦 Restock existing shoes or add new products
- 💰 Automatic discount calculation (5% international, 7% domestic) for bulk orders over 10 items
- 🧾 Saves invoices as .txt files

## Technologies Used
- Python
- File Handling (inventory.txt)
- datetime module

## How to Run
```bash
python SpeedzWear.py
```

## Project Structure
```
speedzwear/
├── SpeedzWear.py      # Main program
└── inventory.txt      # Shoe inventory data
```
